Inimigo com state machine basico.
1. Idle, se move dentro de uma região
2. Começa a perseguir jogador quando ele se aproxima
3. Segue o jogador até que tenha se distanciado demais do centro da região de spawn
4. Retorna para **o centro** da região de spawn quando para de perseguir player
